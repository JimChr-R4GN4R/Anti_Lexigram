===========================
Installation:
1) Run "Setup-x86_64.exe"

2) install packages from the setup:


1) caja-Python

2) Lynx

===========================

After installation complete:

1) Copy "lynx.cfg" from C:\Anti_Lexigram_Project

2) Paste "lynx.cfg" to C:\cygwin64\etc (replace it with the previous one)

3) Done!

===========================

Execution:

1) Open cygwin64 via it's shortcut on Desktop

2) Drag and drop Anti_Lexigram_Project.sh from C:\Anti_Lexigram_Project in the terminal
